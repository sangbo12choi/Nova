# New Star - 새로운 시작

신제품 및 신기술 홍보용 단일 페이지 웹 애플리케이션

## 프로젝트 개요

"New Star - 새로운 시작"은 혁신적인 제품과 최첨단 기술을 소개하는 홍보용 웹사이트입니다. 단일 페이지 애플리케이션(SPA)으로 구현되어 있으며, 현대적이고 반응형인 디자인을 제공합니다.

## 주요 기능

### 1. 히어로 섹션
- 전체 화면 높이의 메인 배너
- 야경 배경 이미지 (50% 투명도 적용)
- 그라데이션 오버레이 효과
- 페이드인 애니메이션
- CTA 버튼

### 2. 소개 섹션
- 3개의 카드 레이아웃
- 혁신적인 제품, 최첨단 기술, 지속가능한 미래 소개
- 호버 효과

### 3. 특징 섹션
- 4개의 주요 특징 카드
- 아이콘 기반 시각화
- 빠른 성능, 보안 강화, 반응형 디자인, 실시간 동기화

### 4. 기술 섹션
- 4개의 핵심 기술 소개
- 글래스모피즘 디자인 효과
- AI & Machine Learning, Cloud Computing, IoT Integration, Blockchain

### 5. 문의 섹션
- 연락 폼 (이름, 이메일, 메시지)
- 폼 제출 핸들러

### 6. 네비게이션
- 고정형 상단 네비게이션 바
- 부드러운 스크롤 애니메이션
- 반응형 디자인

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - Flexbox & Grid Layout
  - CSS Animations
  - Backdrop Filter (글래스모피즘)
  - CSS Variables
- **JavaScript (Vanilla)**: 
  - 스무스 스크롤
  - 폼 제출 처리
  - 스크롤 이벤트 리스너

## 디자인 특징

- **색상 스키마**: 보라색 그라데이션 (#667eea → #764ba2)
- **배경**: 야경 이미지 (Unsplash)
- **애니메이션**: 페이드인 업, 호버 효과, 부드러운 스크롤
- **반응형**: 모바일, 태블릿, 데스크톱 지원

## 파일 구조

```
Nova/
├── index.html          # 메인 웹사이트 파일
├── README.md          # 프로젝트 설명서
└── Report/
    └── 작업결과보고서.md  # 상세 작업 보고서
```

## 사용 방법

### 1. 직접 열기
`index.html` 파일을 웹 브라우저에서 직접 열어 확인할 수 있습니다.

### 2. 로컬 웹 서버 실행 (권장)
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

브라우저에서 `http://localhost:8000` 접속
